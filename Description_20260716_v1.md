# Memory Soundtrack Generator

## Project Description
Google Cloud의 최첨단 멀티모달 및 생성 AI 기술을 결합하여 개인의 사진 데이터로부터 고유한 오디오-비주얼 콘텐츠를 자율적으로 생성하는 웹 애플리케이션입니다. Gemini 3.5 Flash를 통해 다차원적 이미지 스토리텔링과 감성 벡터(Vibe)를 심층 분석하고, 이를 바탕으로 DeepMind의 음악 생성 특화 모델인 Lyria 3를 호출하여 30초 분량의 맞춤형 가사 음원을 작곡합니다. 동시에 Imagen 3 모델로 음악의 정서적 일관성을 투영한 고해상도 앨범 커버 아트를 실시간으로 랜더링하는 혁신적인 멀티모달 오케스트레이션 파이프라인을 구축했습니다.
