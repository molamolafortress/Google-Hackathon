# 🌌 Memory Soundtrack Generator - 실행 가이드 (Execution Guide)

본 프로젝트는 사용자가 업로드한 **추억 사진(시퀀스 이미지)들을 멀티모달 AI(Gemini 3.5 Flash)로 분석하여, 사진에 얽힌 감동적인 스토리라인을 추출하고, 이에 최적화된 맞춤형 배경 음악(Google Lyria) 및 시그니처 앨범 아트(Imagen 4)를 원스톱으로 생성 및 재생하는 프리미엄 웹 애플리케이션**입니다.

이 가이드는 로컬 및 로컬 네트워크(LAN) 환경에서 본 애플리케이션을 설치하고 구동하는 최적의 프로세스를 설명합니다.

---

## 📌 주요 특징 (Key Features)

*   **Premium Ultimate Edition (v4)**: Streamlit 기반의 올인원(One-stop) 소비자용 최종 통합 버전입니다.
*   **Lyria Content Policy Self-Healing**: 아티스트 이름이나 보호 브랜드 노출로 인한 Lyria API 400 차단 예외를 자동으로 감지, 정규식 필터링 및 Gemini 기반 실시간 우회 프롬프트 재합성(Self-healing) 루프로 원활한 동작을 보장합니다.
*   **Aesthetic UI/UX**: 고급스러운 글래스모피즘(Glassmorphism) 스타일 피드, LP판 회전 재생 애니메이션, 크로스페이드 슬라이드쇼, 분할 레이아웃(좌측 이미지 - 우측 스토리 카드)을 구현했습니다.
*   **LAN Exposure (로컬 네트워크 공유)**: 로컬 PC뿐 아니라 동일한 와이파이에 연결된 스마트폰, 태블릿 등에서도 접속 및 테스트를 지원합니다.
*   **40개의 고품질 스토리 테스트 이미지**: 8가지 다양한 라이프스타일 시나리오별로 기획된 테스트 이미지를 즉시 활용해볼 수 있습니다.

## 🚀 빠른 시작 (Quick Start)

### 1. 환경 변수 설정 (`.env`)
프로젝트 루트 디렉토리에 있는 `.env.example` 파일을 복사하여 `.env` 파일을 생성하고, 발급받은 API 키를 입력합니다.

*   Windows PowerShell / CMD 예시:
    ```bash
    copy .env.example .env
    ```
*   `.env` 파일 편집:
    ```env
    # Gemini Developer API Settings
    GEMINI_API_KEY=your_actual_gemini_api_key_here

    # Google Cloud Vertex AI Settings (선택사항)
    ANTIGRAVITY_PROJECT_ID=your_gcp_project_id_here
    ```

### 2. 패키지 설치 (Dependencies)
필요한 Python 패키지를 설치합니다. (가상환경 사용을 권장합니다.)

```bash
# 가상환경 생성 (권장)
python -m venv .venv
# 가상환경 활성화 (Windows)
.venv\Scripts\activate

# 의존성 라이브러리 설치
pip install -r requirements.txt
```
*(만약 requirements.txt가 상위 폴더에 있는 경우, 해당 경로를 올바르게 지정하여 설치해 주십시오.)*

## ⚙️ 실행 방법 (Running the App)

### 방법 A: 배치 파일(.bat) 더블클릭 (가장 간단한 방법 ⚡)
윈도우 탐색기에서 아래 실행기 파일을 **더블클릭**하면 필요한 환경 변수가 연동되며 Streamlit 앱이 자동으로 실행되고 브라우저가 열립니다.

*   **추천 (최종 통합 원스톱 에디션)**: `Run_MemorySoundtrack_Web_20260716_v4.bat`
*   **개발자 샌드박스 에디션 (v7)**: `Run_MemorySoundtrack_20260716_v7.bat`

### 방법 B: CLI 터미널 명령어 직접 실행
터미널에서 직접 실행하려면 아래 명령어를 입력하십시오.

```bash
# 프리미엄 최종 통합 에디션 (Web v4) 실행
streamlit run MemorySoundtrack_Web_20260716_v4.py

# 개발자 샌드박스 에디션 (v7) 실행
streamlit run MemorySoundtrack_20260716_v7.py
```

## 📱 스마트폰/태블릿 등 외부 기기 연결 (LAN Access)

`MemorySoundtrack_Web_20260716_v4.py`는 기동 시 로컬 컴퓨터의 공유기 IP 주소(LAN IP)를 자동으로 감지하여 터미널에 표시해 줍니다.

1.  컴퓨터와 스마트폰을 **동일한 Wi-Fi 공유기**에 연결합니다.
2.  배치 파일 실행 시 터미널 화면에 나타나는 `Network URL`을 스마트폰 브라우저 주소창에 입력합니다.
    *   예시: `http://192.168.0.15:8501` (포트는 환경에 따라 다를 수 있습니다)
3.  모바일 환경에서도 LP판 재생 및 분할 슬라이드 쇼를 터치하여 프리미엄 데모를 감상할 수 있습니다.
